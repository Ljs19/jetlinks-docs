## 配置API
### 操作步骤
1.**登录**Jetlinks物联网平台。</br>
2.点击顶部**系统管理**，在左侧导航栏，选择**API配置**，进入详情页。</br>
3.勾选需要支持对外输出的API服务，然后点击**保存**。</br>
