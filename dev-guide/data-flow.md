## 示例

##### 操作步骤

1.**登录**Jetlinks物联网平台。
2.在左侧导航栏，选择**告警中心>基础配置**，进入详情页。
3.点击页面顶部tab切换至**数据流转**tab页。
4.填写配置信息，然后点击**保存**。
![img](http://doc.jetlinks.cn/assets/img/89.e58c7eca.png)

| 参数      | 说明                         |
| --------- | ---------------------------- |
| kafka地址 | 填写kafka地址，IP地址+端口。 |
| topic     | 填写topic地址。              |
| 状态      | 启用后配置将生效。           |
